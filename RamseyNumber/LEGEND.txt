LEGEND.txt: Description of directories in kakko/RamseyNumber.

brute_force -- computes Ramsey number R(m,n) by brute force
classification -- implementation of Cartan matrix calculation 
cluster_submission -- holds shell submission scripts for Yeti cluster
database_graphs -- text files with unlabelled simple graphs on 5-9 vertices
eigenvalues_of_H -- calculating eigenvalues of H (with theoretical error -- 
					computed on wrong space, part of preliminary exploration)
gap_files -- holds .g files with generators to be fed into GAP for computing
			 Lie algebras
graph_generator -- executable to generate unlabelled simple graphs
				   source: http://users.cecs.anu.edu.au/~bdm/data/graphs.html
miscellaneous -- miscellaneous functions not in use
p_plus_1 -- computing generators E, F for p+1
p_plus_2 -- computing generators E, F for p+2
p_plus_2_multi_op -- computing generators E, F for p+2 and allowing for sequential 
					 repeated E's and F's
p_plus_2_no_repeat -- computing generators E, F for p+2 and removing repeated pairs
					  of subblocks optimization
p_plus_3 -- computing generators E, F for p+3
p_plus_3_multi_op -- computing generators E, F for p+3 and allowing for sequential 
					 repeated E's and F's
p_plus_3_no_repeat -- computing generators E, F for p+3 and removing repeated 
					   pairs of subblocks optimization
random_test -- generates all 0,1-matrices with 0s on the main diagonal for
			   GAP experiments
sl_n_generator -- classification of Lie algebras generated by block diagonal 
				  matrices
